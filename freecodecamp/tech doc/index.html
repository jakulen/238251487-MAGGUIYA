JavaScript

JavaScript is a versatile programming language used for web development.

It allows you to implement complex features on web pages.

let name = "John";

JavaScript can be run on the client side, within the user's browser, or on the server side with Node.js.

const age = 30;

Common JavaScript uses include form validation, dynamic content, animations, and much more.

  • Functions
  • Loops
  • DOM Manipulation
  • Event Handling
Variables and Data Types

JavaScript uses variables to store data.

There are different data types like strings, numbers, booleans, and objects.

let myNumber = 100;

To declare a variable, we can use `let`, `const`, or `var`.

const myString = "Hello World";

Each variable can hold a specific type of value.

  • String
  • Number
  • Boolean
  • Object
  • Array
Functions

Functions are reusable blocks of code that perform a specific task.

In JavaScript, functions can be declared in different ways.

function greet() { console.log("Hello!"); }

You can call a function to execute its code.

greet();

Functions can also return values.

  • Function Declaration
  • Function Expression
  • Arrow Functions
  • Callbacks
  • Higher-order Functions
Loops

Loops allow you to execute a block of code multiple times.

There are several types of loops in JavaScript.

for (let i = 0; i < 5; i++) { console.log(i); }

Loops are used for iterating over arrays, numbers, or other data structures.

while (condition) { // code };

Common loops include `for`, `while`, and `do...while`.

  • for Loop
  • while Loop
  • do...while Loop
  • forEach (Array method)
  • for...in (Object properties)
Objects

Objects are collections of key-value pairs in JavaScript.

They can store data in a structured way.

let person = { name: "John", age: 30 };

You can access object properties using dot notation or bracket notation.

person.name

Objects are essential for representing more complex data structures.

  • Key-Value Pairs
  • Dot Notation
  • Bracket Notation
  • Methods
  • Object Destructuring
